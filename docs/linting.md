# Linting & formatting

Five tools cover four file types. WordPress shareable configs throughout.

## Tools

| Tool                                           | Scope                               | Auto-fix           |
| ---------------------------------------------- | ----------------------------------- | ------------------ |
| ESLint (`@wordpress/eslint-plugin`)            | `*.js`, `*.mjs`                     | `eslint --fix`     |
| Stylelint (`@wordpress/stylelint-config/scss`) | `*.scss`, `*.css`                   | `stylelint --fix`  |
| Prettier (`@wordpress/prettier-config`)        | JS, CSS, SCSS, JSON, MD, HTML, YAML | `prettier --write` |
| PHPCS (`WordPress` standard)                   | `*.php`                             | `phpcbf`           |
| PHPStan (level 5, with WP stubs)               | `*.php`                             | none               |

## Configs

| File                | Tool        | Notes                                                                                |
| ------------------- | ----------- | ------------------------------------------------------------------------------------ |
| `eslint.config.js`  | ESLint      | Flat config, extends WP recommended                                                  |
| `.stylelintrc.json` | Stylelint   | Extends `@wordpress/stylelint-config/scss`; relaxes `no-empty-source` for stub files |
| `.stylelintignore`  | Stylelint   | Excludes auto-generated `src/css/tokens.scss`                                        |
| `.prettierignore`   | Prettier    | Excludes `assets/`, `vendor/`, lockfiles                                             |
| `.editorconfig`     | All editors | Tabs, LF                                                                             |
| `phpcs.xml.dist`    | PHPCS       | `WordPress` minus `WP.I18n`; excludes `vendor/`, `node_modules/`, `assets/`          |
| `phpstan.neon.dist` | PHPStan     | Level 5; loads WP stubs from `szepeviktor/phpstan-wordpress`                         |

## Scripts

| Command                 | Runs                                             |
| ----------------------- | ------------------------------------------------ |
| `pnpm run lint`         | ESLint → Stylelint → PHPCS → PHPStan (fail-fast) |
| `pnpm run lint:js`      | ESLint only                                      |
| `pnpm run lint:css`     | Stylelint only                                   |
| `pnpm run lint:php`     | PHPCS + PHPStan                                  |
| `pnpm run format`       | Prettier `--write` then `phpcbf`                 |
| `pnpm run format:php`   | `phpcbf` only                                    |
| `pnpm run format:check` | Prettier `--check` (read-only)                   |

The Composer commands work standalone: `composer lint`, `composer phpstan`, `composer format`.

## Notes

- The `WordPress.WP.I18n` rule is disabled — this theme doesn't use translations.
- `tokens.scss` is `.stylelintignore`d because its file header says it's auto-generated by `component2block`.
- PHPStan runs with `--memory-limit=1G` because the WP function stubs are large.
