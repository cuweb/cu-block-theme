# Theme structure

## Directory map

```
cu-block-theme/
├── .github/workflows/      # GitHub Actions CI
│   └── ci.yml
├── .husky/                 # Git hooks
│   └── pre-commit
├── assets/                 # Compiled output (do not edit)
│   ├── css/                # styles.css, editor.css, blocks/*.css
│   ├── fonts/inter/        # Self-hosted Inter .woff2 files
│   ├── images/
│   └── js/                 # script.js
├── classes/                # PHP modules (Composer-autoloaded)
│   ├── class-enqueues.php
│   └── class-user-roles.php
├── docs/                   # Documentation
├── parts/                  # Block template parts
│   ├── header.html
│   └── footer.html
├── patterns/               # Block patterns (empty)
├── src/                    # Source files (edit these)
│   ├── styles.scss         # Front-end entry
│   ├── editor.scss         # Editor entry
│   ├── script.js           # JS entry
│   ├── css/                # SCSS partials
│   │   ├── tokens.scss     # Auto-generated design tokens
│   │   ├── base.scss
│   │   ├── layout.scss     # Stub
│   │   ├── utilities.scss  # Stub
│   │   └── blocks/         # Per-block SCSS
│   └── js/
├── styles/                 # Block style variations (JSON)
│   └── buttons/secondary.json
├── templates/              # Block templates
│   ├── index.html
│   ├── page.html
│   └── single.html
├── vendor/                 # Composer dependencies
├── composer.json           # PHP deps + scripts
├── eslint.config.js
├── functions.php           # Bootstrap: autoload + module init
├── package.json
├── phpcs.xml.dist
├── phpstan.neon.dist
├── pnpm-lock.yaml
├── style.css               # Theme metadata header
├── theme.json              # Local theme.json overrides
└── theme-rds.json          # Injected design system base layer
```

## Key files

| File                         | Purpose                                                                                                                                                      |
| ---------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `theme.json`                 | Local overrides on top of `theme-rds.json`. Adds gradients, block styles, registers template parts.                                                          |
| `theme-rds.json`             | Source-of-truth design system: palette, gradients, spacing, fonts, sizes, shadows, layout, element defaults. Loaded via `wp_theme_json_data_default` filter. |
| `functions.php`              | Loads Composer autoload; instantiates module classes.                                                                                                        |
| `style.css`                  | WordPress theme header only.                                                                                                                                 |
| `classes/class-enqueues.php` | Asset enqueueing, per-block stylesheet auto-registration, `theme-rds.json` injection, pattern category registration.                                         |
| `src/css/tokens.scss`        | Design tokens as CSS custom properties (auto-generated by `component2block`).                                                                                |

## Source → output

```
src/styles.scss        ──sass──→  assets/css/styles.css
src/editor.scss        ──sass──→  assets/css/editor.css
src/css/blocks/*.scss  ──sass──→  assets/css/blocks/*.css
src/script.js          ──esbuild──→  assets/js/script.js
```

See [build pipeline](build-pipeline.md).
